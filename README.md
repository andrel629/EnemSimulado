## 📖 Sobre o projeto

O **ENEM Simulator** é uma aplicação web desenvolvida para proporcionar uma experiência semelhante à realização da prova do ENEM.

A plataforma gera simulados automaticamente a partir de uma API de questões, organizando-as por área do conhecimento e apresentando uma questão por vez em uma interface simples, moderna e intuitiva.

Ao final do simulado, o usuário recebe sua pontuação e poderá visualizar um resumo do seu desempenho.

---

## ✨ Funcionalidades

* 📚 Geração automática de simulados
* 📝 Questões organizadas por disciplina
* 🌎 Escolha do idioma (Inglês ou Espanhol)
* 📊 Simulados com 20, 40 ou 80 questões
* ✅ Correção imediata das respostas
* 📈 Cálculo da nota final
* 💾 Armazenamento das respostas utilizando Local Storage

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* Local Storage
* API de Questões do ENEM
