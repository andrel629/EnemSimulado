Acho que essa é uma excelente ideia para portfólio. Ela demonstra integração com API externa, manipulação de estado, interface rica, correção de provas e envio automático de resultados. É um projeto bem mais interessante do que um simples CRUD.

Eu faria algumas mudanças na arquitetura para que o projeto pareça um sistema profissional.
